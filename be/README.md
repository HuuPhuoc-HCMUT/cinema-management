# Cinemax Backend API

Backend API cho hệ thống đặt vé xem phim.

## Cài đặt

```bash
npm install
```

## Cấu hình

1. Copy file `.env.example` thành `.env`
2. Cập nhật thông tin database và JWT secret trong file `.env`

## Khởi tạo Database

```bash
npm run init-db
```

## Chạy server

```bash
# Development
npm run dev

# Production
npm start
```
